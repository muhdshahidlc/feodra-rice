# Minimal Fedora Hyprland Rice

A clean and minimal Hyprland setup inspired by modern Linux rice setups.

## Components

- WM: Hyprland
- Bar: Waybar
- Terminal: Kitty
- Launcher: Rofi
- Notifications: SwayNC
- Logout Menu: Wlogout
- Theme Engine: Wallust
- GTK Theme: GTK 3 / GTK 4
